# my-fitness-plan-frontend

## Project setup
```
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for development
```
npm run build
```

### Compiles and minifies for production
```
npm run build:prod
```

## [部署到 firebase](https://my-fitness-pal-plus.web.app)

```bash
npm run build:dev
firebase deploy
```
